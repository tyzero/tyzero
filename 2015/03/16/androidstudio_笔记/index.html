AndroidStudio 笔记

  1. 用来记录开发安卓时遇到比较坑爹的问题。
  2. 记录需要靠“记”的东西。
  3. 记录靠搜索、文档、拷贝等比较烦又需要经常到的东西。

  4. 随时更新。

    1.解决gradle无法同步远程依赖

    gradle 包含有远程依赖，build的时候报错

    dependencies {
    compile fileTree(dir: ‘libs’, include: [‘.jar’, ‘.aar’])
    compile ‘com.etsy.android.grid:library:1.0.5’
    compile project(‘:extras:ShimmerAndroid’)
    compile ‘com.alexvasilkov:foldable-layout:1.0.1’
    }

出现：

Could not resolve XXX.XXX.XXX:library:1.*.~.
Could not HEAD 'https://jcenter.bintray.comXXX.XXX.XXX:library:1.*.~.pom'.
     > peer not authenticated

解决方法：

替换jcenter为http方式的地址

repositories {
    jcenter {
        url "http://jcenter.bintray.com/"
    }
   //jcenter()
}

2.androidstudio下使用greenDAO生成器

greendao在git上的ExampleDaoGenerator可以正常导入eclipse使用，但是要导入as就要做一些额外的工作才能正常使用。

1.点选项目名，点击F4(keymap–>MacOs)调出project structure

2.点击+增加一个Module，选择module type 为java library，命名为GreenDAoGenerator

3.创建成功后，把ExampleDaoGenerator下的jar复制到GreenDAoGenerator下的lib文件夹，源码也复制到相应是src文件夹。

最后，gradle刷新一下，没有错误就成功了。

2.androidstudio 快捷键

弹出文件结构
OS X: cmd+f12

弹出调用层级
OS X: ctrl+alt+h

伸缩代码块
OS X: alt+plus/minus
这个特性的目的是隐藏在某一时刻你不关心的代码。在这个简单的形式中，它将会隐藏整个代码块（如：当你打开一个新文件时忽略导入列表）。更有趣的是它可以隐藏匿名内部类周围的模板代码让它看起来像是一个lambda表达式。

查找Actions
OS X: cmd+shift+a

上下移动当前行
OS X: alt+shift+up/down
